Star Wars Game

Technologies Used

Summary

  IN one of my computer science classes in High School, we were tasked with coding a game using Object oriented Programming in JavaScript. As a huge fan of Star Wars, I created a Star Wars-themed game where you play as a Rebellion pilot shooting down enemy Tie Fighters. The game uses canvas animations and JavaScript to function. The game even includes music from the movies!
