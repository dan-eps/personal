Kanoodle Solver

Technologies Used

Summary

  Kanoodle is a brain-teasing puzzle game created by Educational Insights, that involves placing 12 unique "noodles" or puzzle pieces onto a 5 by 11 grid. In the summer of 2023, Kanoodle rapidly became a hit with teenagers because of social media, with people playing the game slightly different than the original creators intended. This new method involved picking somewhere between 1-3 pieces, putting them in the kanoodle case, shaking it p, and trying to fill in the remaining slots with the remaining pieces.



While most of these configurations are solvable, some are not and often it is difficult to find a solution or determine if a solution is even possible. After watching my girlfriend fall in love with the game, I decided this was a problem I could fix. I wrote a script that displays a user interface to allow the user to input the setup of the game, and then it runs an exhaustive algorithm to find any and all solutions.



Here is a brief look at that algorithm:
    
    def tryPlace(pieces, opens):

        # Next piece to place is the first piece in pieces
        p = pieces[0]

        # For each orientation
        for j in range(p.flips):
            for i in range(p.rots):

                # Copy the opens to a local copy for the current piece
                pc = opens.copy()
                for c in pc:

                    # For each open spot, 
                    # 1 - move to the open spot,
                    # 2 - if that is a valid placement, place the piece, then tryPlace next piece
                    # 3 - remove the piece and try next spot

                            temp = opens.copy()
                            for t in p.shape:
                                temp.remove(t)
                            # Place the next piece
                            tryPlace(pieces[1:], temp)

                p.rotate90()
            # Flip the piece to try that solution
            p.flip()
